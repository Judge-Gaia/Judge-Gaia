# Interim Report 01 Guide

## 보고서 제목

중간 보고서 1: 풀스택 인프라 구축 및 MVP 기반 마련.

## 진행률

20%.

## 집중 공략 포인트

3D 지구가 렌더링된 홈 화면의 시각 자료를 캡처해 첨부한다. 백엔드 세팅 내용도 포함하되, 다음 메시지를 강조한다.

> 사용자에게 시각적 흥미를 주기 위해 Three.js를 활용한 초기 뷰를 구성했다.

## 필수 포함 항목

- 추가된 기능 요약.
- 진행률: 20%.
- 프론트엔드 코드 주석 내용.
- 백엔드 코드 주석 내용.
- 3D 지구 홈 화면 캡처.
- 서버/DB/Socket.IO 세팅 요약.

## 기능 요약 템플릿

```markdown
### 추가된 기능 요약

- React + Vite 기반 클라이언트 프로젝트를 구성했다.
- Three.js로 화면 중앙에 3D 지구 초기 뷰를 렌더링했다.
- 닉네임 입력 후 개인 세션으로 즉시 입장하는 홈 화면 흐름을 구성했다.
- Fastify 기반 Node.js 서버와 Socket.IO 실시간 연결 기반을 구성했다.
- PostgreSQL 유저/랭킹 스키마를 추가했다.
- 클라이언트 입장 시 서버가 고유 세션 ID를 발급하도록 구성했다.

### 진행률

- 전체 MVP 대비 20%.

### 강조 포인트

사용자에게 시각적 흥미를 주기 위해 Three.js를 활용한 초기 뷰를 구성했다.
```

## 캡처 체크리스트

- 홈 화면에 프로젝트명 또는 가이아 테마가 보인다.
- 3D 지구가 중앙에서 명확하게 보인다.
- 닉네임 입력 UI가 보인다.
- SDGs 13/14/15 메시지가 보인다.
- 화면이 보고서에 넣기 좋게 깨지지 않는다.
