# Project Brief

## 프로젝트명

가이아 심판자, 임시 영문명 `GAIA: Judgment Loop`.

## 제품 개요

- 장르: 실시간 웹 io 액션 + 환경 이슈 학습형 업적 수집.
- 콘셉트: 플레이어는 신적 관찰자로서 지구 곳곳의 환경 파괴 이벤트를 감지하고, 재난을 정화한 뒤 원인 NPC에게 상징적 제재를 수행한다.
- 핵심 테마: SDGs 13 기후변화, SDGs 14 해양생태계, SDGs 15 육상생태계.
- 표현 원칙: 현실 인물 대상 폭력은 금지한다. 모든 대상은 가상 NPC 데이터이며, 제재 연출은 비폭력적이고 상징적으로 표현한다.

## 목표

- 짧은 플레이 구간에서 즉각적 재미와 학습 정보를 동시에 제공한다.
- 업적과 랭킹 루프로 반복 플레이를 유도한다.
- URL 접속 후 빠르게 플레이 가능한 웹 접근성을 확보한다.

## 타깃

- 12세 이상 캐주얼/미드코어 웹 유저.
- 짧은 세션과 즉시 시작, 순위 경쟁을 선호하는 io 게임 유저.
- 환경 주제에 흥미가 있는 유저.

## 개발 원칙

- MVP는 계정 없이 닉네임만으로 즉시 시작한다.
- 클라이언트 점수는 신뢰하지 않고, 서버가 이벤트 완료/점수/업적/랭킹을 판정한다.
- 시각적 흥미는 Three.js 기반 3D 지구 뷰에서 먼저 확보한다.
- 학습 정보는 업적 카드와 SDGs 메시지로 자연스럽게 노출한다.
