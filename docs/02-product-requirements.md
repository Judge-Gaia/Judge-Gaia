# Product Requirements

## 핵심 게임 루프

1. 입장: 닉네임 입력.
2. 시작: 매칭 없이 즉시 개인 세션 생성.
3. 진행: 지구 뷰에서 환경 이벤트 알림 발생.
4. 진입: 이벤트 클릭 시 카메라 줌인으로 현장 이동.
5. 대응: 재난 정화, 원인 NPC 식별, 상징적 제재.
6. 보상: 업적 카드 팝업으로 SDG, 문제, 제도, 사례 정보를 제공.
7. 반복: 다음 이벤트 대응.
8. 종료: 총점, 랭킹, 획득 업적 표시.

## 게임 모드

### 기본 모드

- 주요 이벤트 정화와 필수 업적 달성이 목표.
- 균형형 난이도와 학습 카드 중심의 플레이.

### 울트라 모드

- 고난도 이벤트 연속 대응과 고급 업적/점수 극대화가 목표.
- 시작 전 도구 선택, 추가 업적 등급, 재화 기반 해금 요소를 포함한다.
- MVP에서는 라이트 버전으로 도구 선택과 추가 보상만 우선 구현한다.

## 월드/카메라

- 3D 지구 회전, 확대, 축소.
- 이벤트 핀 표시와 우선순위별 색상.
- 지구 뷰에서 현장 뷰로 전환하는 카메라 애니메이션.

## 이벤트 시스템

- 시간, 난이도, 지역 기반 이벤트 스폰.
- 동시 다발 이벤트 처리.
- 방치 시 피해 확산과 점수 패널티.
- MVP 목표: SDGs 13/14/15별 이벤트 2개씩 총 6개.

## 플레이어 능력

- 정화 능력: 비 내리기, 바람 제어, 해양 정화 파동.
- 제재 능력:
  - 봉인: 대상 NPC 행동을 일정 시간 봉쇄.
  - 전기: 대상 NPC 작업/이동 효율 저하. 비폭력 연출로 제한.
  - 우주 추방: 대상 NPC를 전장 외 격리 구역으로 일시 이동.
- 자원 규칙: 신의 힘 바, 쿨다운, 범위, 세기.

## NPC/탐색

- 원인 NPC 단서 시스템: 행동 로그, 흔적, 위치 이동.
- 오탐/오판 시 감점.
- 현실 인물이나 실제 개인을 대상으로 삼지 않는다.

## UI/UX 레이아웃

- 중앙: 3D 월드.
- 중앙 하단: 신의 힘 바, 스킬 슬롯, 쿨다운.
- 우측 상단에서 하단: 업적 카드 피드.
- 좌측 상단: 현재 목표와 진행도.
- 좌측 하단: 미니 랭킹과 미니 로그.
- 플레이 종료 모달: 점수, 기여도, 업적, 재시작.

## 업적/학습 카드

필수 필드:

- 업적명.
- SDG 분류: 13, 14, 15.
- 문제 설명: 왜 위험한지.
- 제도/법적 포인트: 일반화된 정책 관점.
- 실제 유사 사례 요약.
- 보상: 점수, 재화, 칭호.

저장 요구:

- 업적 보관함.
- 컬렉션 완성도 표시.
- 히든 업적과 이스터에그.

## 점수/랭킹

점수 구성:

- 정화 속도.
- 원인 식별 정확도.
- 제재 정확도.
- 연속 성공 보너스.

랭킹 요구:

- 비동기 글로벌 리더보드.
- 모드별 랭킹 분리: 기본, 울트라.
- 서버 기준 점수만 랭킹에 반영.

## 네트워크/치트 방지

- MVP 세션 구조: 싱글 플레이 인스턴스 기반.
- 서버 권위 판정: 이벤트 완료, 점수 계산, 업적 지급, 랭킹 반영.
- 클라이언트 점수는 무신뢰.
- 비정상 패턴 탐지 후보: 초당 입력, 텔레포트, 쿨다운 위반.
- 리더보드는 이벤트 또는 주기 기반으로 동기화.

## 성능 요구사항

- 목표 FPS: 60, 최저 30.
- 초기 진입: 광대역 기준 핵심 인터랙션 가능 시점 5초 이내.
- 최적화 후보: LOD, 오브젝트 풀링, KTX2 텍스처 압축, 파티클 품질 자동 조정, 저사양 모드 토글.
